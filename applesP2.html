Explaining How I Used Display: Grid

Hello Mr. Fernandez! If you are currently viewing this, it is probably because you asked me how I used display: grid. I made this little mini skillset to explain just what I used, since the display grid has lots and lots of characteristics.

In this example, I pretty much recreated my own sushi PWYS but in an easier way that can stack since it doesn't use display flex.

Each photo and text is in a div, making 3 divs. The 3 divs are inside of a section, with a class called grid. In my style section for the grid class, these are the characteristics I added.

display: grid;
grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
gap: 10px;
width: 100%;
text-align: center;

First, display: grid; provides you with a singular column, which you can then define.

grid-template-columns allows you to create a template of each column, essentially defining the size of each column. I used repeat which makes it so that the pattern repeats for each div continuously. I used auto-fit, which will expand the grid items to fill the available space, not to be confused with auto-fill, which will keep the available space reserved without altering the grid items width.

I used minmax which basically makes the page responsive to when you resize the page or open it on a phone. The minmax I used sets the minimum to 100px, and the maximum to 1fr. fr stands for fraction, which sets the maximum size of the column to a fraction of the available space in the grid container, using 1 to make it take up the entirety of its container.

Using minmax and auto-fit (or auto-fill if you choose to) are both useful because it makes the display responsive, meaning that it will automatically wrap itself under or stack itself when the page gets smaller.
